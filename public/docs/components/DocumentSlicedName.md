# DocumentSlicedName

> Display a document name in a sliced maner (to include parents).

## Props

| Prop name       | Description                     | Type    | Values | Default |
| --------------- | ------------------------------- | ------- | ------ | ------- |
| document        | The selected document           | object  | -      |         |
| interactiveRoot | Document's root should be links | boolean | -      |         |
