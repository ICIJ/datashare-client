# ProjectSelector

> A single-project selector input.

## Props

| Prop name | Description                 | Type   | Values           | Default |
| --------- | --------------------------- | ------ | ---------------- | ------- |
| v-model   | The selected project value. | string | -                |         |
| size      | Select size                 | string | `sm`, `md`, `lg` | 'md'    |
