# EmailString

> Rich email display with shortcuts to the global search.

## Props

| Prop name | Description                                                | Type   | Values | Default |
| --------- | ---------------------------------------------------------- | ------ | ------ | ------- |
| email     | Email to display.                                          | string | -      |         |
| tag       | Specify the HTML tag to render instead of the default tag. | string | -      | 'span'  |
